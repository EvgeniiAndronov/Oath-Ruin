### Задача 5: Синхронизация перемещения
**ID:** SPR1-05  
**Оценка:** 6 часов  
**Статус:** ⏳ Ожидает

**Чеклист:**
- [ ] Клиент отправляет `move_character`
- [ ] Сервер валидирует ход
- [ ] Сервер рассылает обновление всем
- [ ] Другие клиенты обновляют состояние

**Протокол:**
```json
// Клиент → Сервер
{
  "type": "move_character",
  "payload": {
    "character_id": "char_1",
    "x": 5,
    "y": 3
  }
}

// Сервер → Все
{
  "type": "game_state",
  "payload": {
    "characters": [...]
  }
}
```

**Критерии готовности:**
- Запусти 2 клиента
- Перемести персонажа в одном
- Во втором видно обновление позиции
